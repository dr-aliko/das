02 · Favicons

Browser tab

Modern browsers will use favicon.svg (which adapts to dark mode automatically). Raster fallbacks shipped for legacy environments.

03 · PWA & mobile

Home Screen icons

Standard icons render with rounded corners as-shipped. Maskable icons fill the canvas — the OS clips them to circle/squircle/whatever shape the device prefers. Both ship at 192 and 512.

Why two flavors? Standard icons assume the OS shows them as-is. Maskable icons assume the OS will clip them — so they need a full-bleed background and the mark sized inside the inner 80% "safe zone." Above, you can see the same maskable rendered as a circle (Android Adaptive) and a squircle (PWA) — the mark stays comfortably visible in both.

07 · Integration

Drop-in checklist

Full instructions in USAGE.md. Quick version:

  1. Copy vagus-brand/ into static/vagus/. Run collectstatic.
  2. Add the favicon + apple-touch-icon + manifest links from USAGE.md §2 to base.html <head>.
  3. Add <meta name="theme-color" content="#6366F1">.
  4. Add OG / Twitter card meta tags pointing at og-card.png (use absolute URL).
  5. Replace the old DAS wordmark in the navbar with {% include "vagus/svg/vagus-mark.svg" %} + "Vagus" text.
